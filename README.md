# Privacy Policy — TubeFusion

_Last updated: June 2026_

---

## Overview

TubeFusion is a Chrome extension that extracts YouTube video transcripts and uses Google's Gemini API to generate content assets. This policy explains exactly what data is used, where it goes, and what is stored.

**TubeFusion has no backend server. No data is ever sent to or stored by the developer.**

---

## What is used

### 1. YouTube Transcript
Your YouTube transcript is sent directly to Google’s Gemini API to generate content. TubeFusion never sees or stores it

### 2. Gemini API Key
Your Gemini API key and preferences are stored locally in your own Chrome browser via chrome.storage.sync.

### 3. Workspace State
Generated content and workspace data are stored locally on your device via chrome.storage.local and cleared when you reset the workspace

---

## Third parties

Your transcript is processed by Google’s Gemini API, governed by [Google's Privacy Policy](https://policies.google.com/privacy).

---

## Changes to This Policy

If the extension is updated in a way that changes how data is handled, this policy will be updated and the date above will reflect the change.
